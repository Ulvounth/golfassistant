# Frontend

React application for golf round tracking.

## Tech

React 18, TypeScript, Vite, Tailwind CSS, React Router, Axios

## Setup

```bash
npm install

# Create .env file:
VITE_API_URL=http://localhost:3001/api

npm run dev
```

## Features

- User registration and login
- Round registration (18-hole and 9-hole)
- Course management
- Dashboard with statistics
- Leaderboard
- Profile page

## Build

```bash
npm run build
```
